# 日课一问 (Daily Question)：每日一问，破局人生

最新版仓库地址：https://github.com/cnfeat/DailyQuestion


## 1、🎉简介


《论语·学而》曾子曰：「吾日三省吾身。」在数字时代，我们该如何找回这种修身智慧？

这个「日课一问」浏览器插件可以轻松实现。每天我们会无数次打开浏览器新标签页，但大多是在下意识地刷网页。安装后，新标签页将不再是空白或广告，而是变成一个引发你深度思考的问题卡片，让你不经意强制刺破日常惯性，助你激发深度思考，成为自己的破壁者。

## 2、🎶核心特性

### 开启深度反思
内置 365+ 个引发深思的底层问题。在每一次打开新标签页的瞬间，用极简卡片进行灵魂质询，将碎片时间重构成反思时刻。

### 专注当下破局
效法古人「三省」之数，每日限定抽取 3 张思考卡片。支持「换一问」自主选择，抽满后进入「循环查看」模式，引导深度咀嚼。

### 唤醒时间感知
实时同步全年 365 格进度矩阵，显示生命流逝百分比。用冷峻的刻度营造紧迫感，提醒你抓紧时间行动，不再对光阴麻木。

### 沉淀思想资产
极致简约设计，支持一键生成高清问题海报。让你的深度反思具备视觉质感，方便保存或分享，记录自我进化的轨迹。

## 3、📍安装指南

### 在线安装

Chrome 应用商店地址：


### 离线安装


通过开发者模式进行本地加载：

1. 将本仓库的所有文件下载并解压到一个独立的文件夹中。
2. 在 Chrome 浏览器的地址栏输入 `chrome://extensions/` 并回车。
3. 在页面右上角，打开「开发者模式」开关。
4. 点击左上角的「加载已解压的扩展程序」按钮，选择刚才解压的文件夹。
5. 打开一个全新的标签页，体验这套极简的日课一问


## 4、🎈为什么发起这个项目


### 1

大多数人一生中，从未真正为自己思考过一天。

每天去上毫无意义的班，在无尽的消费中麻痹，直到再次疲惫地睡去。你以为你拥有自由意志，实际上，你只是被算法、欲望和别人的期待推着走。

如何打破这个死循环？曾子说「吾日三省吾身」。但在诱惑满盈的今天，仅靠意志力是不够的，你需要一套系统。

### 2

在这个时代，学会思考从未如此重要。 而思考的起点，是提问。 你的提问能力，决定了你人生的结局。 那些坚持练习提问的人，最终会获胜。

但人天生是懒惰的。 你需要系统。 成功者最大的秘密，从来都不是天赋异禀。 成为更好的自己，并非靠意志力，而是靠设计系统。

那么，如何设计一个让自己无法逃避的思考系统？

### 3

我从 2020 年开始践行自我反思。 每天直面一个我一直在逃避的灵魂拷问——

+ 如果你早就去做那些害怕的事情，现在的生活会是怎样？
+ 如果十年后，你要给理想的自己写一段颁奖辞，你会怎么写？
+ 对于你现在努力要完成的重要任务，有用且有价值的最小进步是什么？
+ 你是否在遇到大量问题时行动瘫痪？事后有没有做任何哪怕微小的努力来改变现状？

这些问题像一记当头棒喝，强迫你跳出固有的思维模式。 它们曾让我从当局者迷的井底中爬出，渡过人生的难关。 切身实践四年来，我不再依赖外界的碎片化信息投喂，而是通过这些问题直接触达内心。

你敢不敢用一年的时间，彻底重构你的人生？ 让自己，成为自己的破壁者。

为了将这种刻意练习固化，我将精选的问题整理成「日课一问」卡片系统，做成了一个浏览器新标签页插件。每天打开电脑，它都会强制你直面今天的拷问。

我还特意在卡片底部，设计了 365 格的时间方块矩阵。 它在无声地提醒你，不要虚度光阴。将每日的自己，都当成一个可以不断优化的系统。 

用这些日积月累的自我解答，自己成为自己的破壁者，完成人生底盘的重塑。

### 4

并不是所有问题，都需要写下答案。也并不是所有问题，都有标准答案。

思考的过程本身，就是最大的启发。更多时候，我喜欢将问题挂在大脑的后台运行，以时间做缓存，来持续读取自己。

渐渐地，你在训练自己主动构思并回答问题的能力。 变成一个能自己出题，并自己解决难题的人。 掌握这种能力，远远比得到任何高人的指点都来得重要。

有了这套系统，你将可以：

+ 主动驱动思考，防止被动逃避。
+ 激发好奇探索，每日自省反思。
+ 持续读取自我，渐进底层改变。
+ 定期深入反思，持续自我精进。

你可以继续假装清醒地沉睡。 也可以每天用一个好问题，破局人生。

愿每日都有好问题与你同在。
