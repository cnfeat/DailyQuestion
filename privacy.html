日课一问 | 隐私权政策

最近更新日期：2026年5月21日

「日课一问」（Daily Question）是一款注重隐私的浏览器扩展程序。本政策旨在说明我们如何处理用户数据。

1. 数据收集与使用

本扩展程序是一个纯本地运行的工具。我们不会收集、存储或传输您的任何个人身份信息、浏览历史或敏感数据。

2. 本地存储声明

为了实现核心功能，扩展程序会使用浏览器内置的 localStorage 在您的设备本地存储以下信息：

  • 年度进度矩阵的状态（已完成的天数计数）。
  • 当前抽取的问题 ID 和浏览记录。
  • 用户简单的偏好设置。

这些数据仅存储在您的本地浏览器中，不会上传到任何服务器。

3. 权限说明

本扩展程序不需要任何特殊浏览器权限。所有数据仅在您本地浏览器中处理和存储。

4. 政策修改

如果我们将来对数据处理方式做出调整，将通过更新本页面进行告知。

5. 联系方式

如有任何疑问，请通过 GitHub 仓库（https://github.com/cnfeat/DailyQuestion）联系开发者。
